## UYeong Store (Rebuilding...)

스토어를 운영하기 위한 필요한 기능들을 구현해 보고자 제작하게 되었습니다.

<br>

## Skills

- React.js
- Next.js
- MongoDB
- Bootstrap

<br>

## Deployment

- [vercel](https://vercel.com)

<br>
